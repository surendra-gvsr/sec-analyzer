# Frontend Rules

- Always use shadcn/ui components, never build from scratch
- Available components: button, input, card, form, label, sonner
- Tailwind classes only, no inline styles
- Mobile first, base width 375px
- Nova color tokens for all colors
- Every interactive element needs hover and focus state
- Layout first: define structure before building components
- Spacing scale: 4, 8, 12, 16, 24, 32, 48, 64px only
- Max 2 font families, max 3 type sizes per page
